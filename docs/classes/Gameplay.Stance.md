## Methods

### play <Score text="play" /> 

• **play**(): `boolean` 

播放这个姿态对象, 并返回执行结果


::: warning Precautions

这个操作是否会自动同步多端, 取决于调用Character.loadStance()时传入的sync参数.
返回的执行结果, 在不自动同步时多数情况下可靠;
在自动同步时完全不可靠, 总是会返回true, 因为无法判断其他端的执行情况

:::

#### Returns

`boolean`

是否执行成功
